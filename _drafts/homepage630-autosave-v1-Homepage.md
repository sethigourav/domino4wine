<h3 style="color: #6b0031;">
  What is<span class="Apple-converted-space"> </span><a href="https://www.winehq.org/" target="_blank">WINE</a>?
</h3>

\[/vc\_column\_text\]\[vc\_row\_inner\]\[vc\_column\_inner width=&#8221;2/3&#8243;\]\[vc\_column\_text\]

<p style="text-align: justify;">
  <a href="https://www.winehq.org/" target="_blank">Wine</a><span class="Apple-converted-space"> </span>(originally an acronym for &#8220;Wine Is Not an Emulator&#8221;) is a compatibility layer capable of running Windows applications on several POSIX-compliant operating systems, such as Linux, Mac OS X, & BSD. Instead of simulating internal Windows logic like a virtual machine or emulator,<a href="https://www.winehq.org/" target="_blank">Wine</a><span class="Apple-converted-space"> </span>translates Windows API calls into POSIX calls on-the-fly, eliminating the performance and memory penalties of other methods and allowing you to cleanly integrate Windows applications into your desktop.
</p>

<h3 style="color: #6b0031;">
  Why <a href="https://www.codeweavers.com/products/windowsmac/" target="_blank">CrossOver</a>?
</h3>

\[/vc\_column\_text\]\[vc\_row\_inner\]\[vc\_column\_inner width=&#8221;2/3&#8243;\]\[vc\_column\_text\]

<p style="text-align: justify;">
  CrossOver, while composed primarily of WINE open-source core, also contains proprietary &#8220;value-add&#8221; components. CrossOver gives you the ability to run multiple INDEPENDENT copies of the Notes Client, Domino Administrator, and Domino Designer simultaneously &#8212; all while using separate Notes IDs &#8212; with much lower overhead and licensing costs than any available virtualization technology. CrossOver does not require an underlying Windows license and provides less CPU overhead, and therefore a much longer battery life.
</p>

<h6 style="text-align: center; margin-bottom: 10px;">
  Once you received the Email with the download links, proceed with the following steps:
</h6>

<div style="font-size: 90%;">
  <ul>
    <li>
      Unzip, Launch and move CrossOver to Application folder if prompted
    </li>
    <li>
      Create a new Windows XP Bottle by clicking on the bottles icon at the top left of CrossOver, then click the + icon at the bottom left and select New Bottle.
    </li>
    <li>
      Configure the newly created Bottle: <ul>
        <li>
          Right-click the created Bottle and enable<span class="Apple-converted-space"> </span><strong>Legacy X Windows</strong><span class="Apple-converted-space"> </span>from the<strong>Settings</strong><span class="Apple-converted-space"> </span>submenu
        </li>
        <li>
          Reboot the Bottle when prompted
        </li>
        <li>
          <strong>Run Command</strong><span class="Apple-converted-space"> </span>&#8220;regedit&#8221;
        </li>
        <li>
          Navigate to HKEY_CURRENT_USER/Software/Wine/X11 Driver/
        </li>
        <li>
          In the right panel, right-click and create a new String Value and name it<strong>&#8220;ClientSideGraphics&#8221;</strong>
        </li>
        <li>
          Right-click on the newly created key and modify the value to<span class="Apple-converted-space"> </span><strong>&#8220;N&#8221;</strong>
        </li>
      </ul>
    </li>
    
    <li>
      Install the prerequisite applications: <ol>
        <li>
          <strong>Install Windows Application</strong>, scroll to<span class="Apple-converted-space"> </span><strong>&#8220;Runtime Support Components&#8221;</strong>dropdown
        </li>
        <li>
          Select<span class="Apple-converted-space"> </span><strong>&#8220;Microsoft Visual C++ 6.0&#8221;</strong><span class="Apple-converted-space"> </span>and Proceed
        </li>
        <li>
          Select your newly created Bottle and finish the installation process
        </li>
      </ol>
    </li>
    
    <li>
      Repeat the above steps for the following: <ul>
        <li>
          <strong>Platform SDK Redistributable: GDI+</strong>
        </li>
        <li>
          <strong>Internet Explorer 7. Install the XML Parser when prompted</strong>
        </li>
        <li>
          <strong>Flash Player 11</strong>
        </li>
      </ul>
    </li>
    
    <li>
      Install IBM Notes Software: <ul>
        <li>
          Click<span class="Apple-converted-space"> </span><strong>Install Windows Application</strong>
        </li>
        <li>
          Select<span class="Apple-converted-space"> </span><strong>Other Application</strong><span class="Apple-converted-space"> </span>from the<span class="Apple-converted-space"> </span><strong>Unsupported Applications</strong><span class="Apple-converted-space"> </span>dropdown list
        </li>
        <li>
          Select<span class="Apple-converted-space"> </span><strong>&#8220;Choose installer File&#8221;</strong><span class="Apple-converted-space"> </span>browse to your Notes Client (Designer and Administrator) Installer
        </li>
      </ul>
    </li>
    
    <li>
      (If you are experiencing graphics artifacts Launch<span class="Apple-converted-space"> </span><strong>Wine Configuration</strong><span class="Apple-converted-space"> </span>and check<strong>Emulate Virtual Desktop</strong><span class="Apple-converted-space"> </span>in the<span class="Apple-converted-space"> </span><strong>Graphics</strong><span class="Apple-converted-space"> </span>tab).
    </li>
  </ul>

    Once your trial version expires, please register your CrossOver full version using promo code &#8220;DOMINO&#8221; to support this project.
  </p>
